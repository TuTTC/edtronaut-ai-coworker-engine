# Competency Framework — Implementation Guide

## Classification: Internal — HR Leadership Team

### 1. Framework Overview

The Gucci Group Leadership Competency Framework provides a structured approach to assessing, developing, and deploying leadership talent across our 9 brands. It consists of **4 themes**, each with **3 levels** of behavioral expectations.

### 2. Design Principles

1. **Universal but adaptable** — The 4 themes apply to all leaders. The behavior indicators are starting points that each brand can customize ±20%.
2. **Developmental, not evaluative** — The primary purpose is growth, not ranking.
3. **Evidence-based** — Behavior indicators describe observable actions, not personality traits.
4. **Progressive** — Each level builds on the previous. Senior leaders exhibit all Associate and Mid behaviors plus Senior-specific ones.

### 3. Implementation Guidelines

#### 3.1 Calibration Sessions
- Each brand should run calibration sessions to align on what "good" looks like for each indicator in their context.
- Calibration panels should include: Brand CEO, HR Director, 2-3 senior leaders.

#### 3.2 Assessment Methodology
- **Self-assessment** (mandatory) — 10-15 minutes per theme
- **Manager assessment** (mandatory) — conducted during performance review
- **Peer feedback** (optional) — 2-3 peers from same brand
- **Cross-brand feedback** (optional) — for mobility candidates

#### 3.3 Data Handling
- All assessment data is owned by Group HR
- Individual results are confidential between the employee, their manager, and HR
- Aggregate data (anonymized) is shared with brand leadership for development planning
- GDPR compliance required for all European data processing

### 4. Common Pitfalls to Avoid

- **Cookie-cutter application**: Don't use the same workshop slides for Gucci and Qeelin — context matters
- **Assessment fatigue**: Keep the process light — no more than 2 hours total per assessment cycle
- **Neglecting the "why"**: Always connect the framework back to business outcomes
- **Level conflation**: Ensure clear gaps between Associate, Mid, and Senior expectations
